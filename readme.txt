# 🦊 LYNX Task Manager

**Autor:** Franciszek Niemyński  
**Technologie:** Python, SQLite, CustomTkinter  

Prosty menedżer zadań do organizacji dnia.  
Pozwala zapisywać zadania z priorytetami i godziną wykonania, a dane zapisuje lokalnie w SQLite.  
Aplikacja działa offline i automatycznie aktualizuje strukturę bazy danych.

## Funkcje:
- ✅ Dodawanie / usuwanie / przeglądanie zadań  
- ⏰ Ustalanie godziny wykonania  
- 🔥 Kolorowe oznaczenia priorytetów  
- 🧠 Automatyczna naprawa bazy danych
